
This project is a Flask web app that lets users query Australian weather data 
## Quick  Instructionss

1. Create and activate a virtual environment:
```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install dependencies:
```bash
python3 -m pip install -r requirements_phase9.txt
```

3. Run the app:
```bash
python3 run_phase9.py
```

4. Open:
- `http://127.0.0.1:5000`

If the page shows `Prediction model unavailable`, reinstall dependencies in a clean virtual environment and restart the app.
